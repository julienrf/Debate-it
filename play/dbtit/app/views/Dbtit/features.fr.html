Fonctionnalités

Debate it! est conçu pour faciliter le débat.

Disposition innovante

Les réponses à un paragraphe d’un message s’intercale au milieu de ce dernier, afin que les réponses soient toujours dans leur contexte. Citer devient inutile !

Syntaxe simple de mise en forme

Les messages peuvent contenir des éléments essentiels de mise en forme (accentuation, liste, lien hypertexte, etc.)

Gestion des notes de bas de page

Les notes de bas de page peuvent servir à contenir des liens vers les sources appuyant un argument, ou bien un détail explicatif qui serait trop long pour être dans le message initial.

Embranchements de discussions

Si les réponses à un message constituent une digression, il est possible de démarrer une nouvelle discussion à partir de la discussion existante.
